<ul>
<li><strong>ATP-PC System (Phosphogen System)</strong> &#8211; This system is used only for very short durations of up to 10 seconds. The ATP-PC system neither uses <a title="Oxygen" href="/wiki/Oxygen">oxygen</a> nor produces <a title="Lactic acid" href="/wiki/Lactic_acid">lactic acid</a> if oxygen is unavailable and is thus said to be alactic anaerobic. This is the primary system behind very short, powerful movements like a golf swing, a 100 m sprint or powerlifting.</li>
</ul>
<ul>
<li><strong>Anaerobic System (Lactic Acid System)</strong> &#8211; Predominates in supplying energy for exercises lasting less than 2 minutes. Also known as the Glycolytic System. An example of an activity of the intensity and duration that this system works under would be a 400 m sprint.</li>
</ul>
<p style="text-align:center;"><img class="aligncenter" src="http://startedwithscifi.files.wordpress.com/2011/05/weight-lifter.jpg?w=500&h=321" alt="" width="500" height="321" /></p>

<li><strong>Aerobic System</strong> &#8211; This is the long duration energy system. By 5 minutes of exercise the O<sub>2</sub> system is clearly the dominant system. In a 1 km run, this system is already providing approximately half the energy; in a <a title="Marathon" href="/wiki/Marathon">marathon</a> run it provides 98% or more.<sup><a href="#cite_note-0">[1]</a></sup></li>
</ul>
